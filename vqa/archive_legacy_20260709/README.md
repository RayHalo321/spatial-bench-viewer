# Legacy VQA Records

These files belong to the previous VQA report and scoring protocol. They are retained only for traceability and must not be compared directly with the current report.

The public `vqa_report.html` now uses the 2026-07-10 protocol: presence gating, direct `between` questions, positive/inverse relation checks, case normalization, and `20% presence + 80% relation` combined scoring.
